Linux

Install with 'aptitude' (do not install the latest version):

If you're on Debian you can use the aptitude package manager typing the following line of command in your terminal:

1.First we going to update the package repositories:

$ sudo apt-get update

2.Next we need to install some dependancies:

$ sudo apt-get install build-essential zlib1g-dev libncurses5-dev libgdbm-dev libnss3-dev libssl-dev libreadline-dev libffi-dev libsqlite3-dev wget libbz2-dev

3.Now install python:

$ sudo apt-get install python3

Download with 'wget' and compile binary with 'make'(install the latest version):

1.Download the binary file from the offical python download page using 'wget':

'wget' is a package used to download ressources from an http server with the terminal.

$ wget https://www.python.org/ftp/python/3.12.4/Python-3.12.4.tgz

2.Uncompress the tarball using the following command line:

'Tar' is tool to gather and compress file in one archive called 'tarball'.

$ tar -xvf Python-3.12.4.tgz

3.Use 'cd' command to change the current working directory to the 'python3' folder:

$ cd python-3.11.1

4.Check if all dependencies to install python 3.11 are ok:

$ sudo ./configure --enable-optimizations

5.Run the make command to compile the source code:

$ sudo make -j 2

6.Now we can install python:

$ sudo make altinstall

Hello world!

In this section we gonna make our first program in python and review some basic general topic of programing language.

This code below print the message 'hello world' on screen:

Code:

print("hello ,world!)

Output:

>hello, world!


The first word is a function called print().It is used to print text on standard output which is the screen here.
The text between quote is called a string of characters 'hello world'.
A string of charactes can contain letter,number and symbol.

You can also print others values like the result of a math expression:

print( 4 + 2) >6

As you notice , the expression in the code above 4 + 2 has no quotes despite being surrounded by parantheses.The reason is the content of the data that we want to print.

In the first example we printed a string of characters, which must be between quotes whereas in the second example we printed a integer value.

During the execution, we can see that the program print the value 6 which is the result of 4 + 2 and not the equation itself .Python do the addition first and then print the result.

You can use several types of operators for math int your print functionand it will be printed when you execute the code as long it has no quotes.
